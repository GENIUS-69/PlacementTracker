🧠 Tech Stack: Placement Prep Tracker
1. 🏗️ Architecture Style
✅ Monolith (for now)
One backend (Node + Express)
One frontend (React)

👉 Why:

Faster to build
Easier to debug
Enough for single-user localhost
2. 🎨 Frontend Stack
⚛️ Core
React (with Hooks)
Vite (faster than CRA)
🎨 Styling
Tailwind CSS

👉 Why:

Rapid UI (you’ll build dashboard fast)
Clean + modern look (important for resume)
📊 Visualization
Recharts

Used for:

Pie chart (progress)
Bar chart (weekly activity)
Line chart (test improvement)
🧠 State Management
Zustand

👉 Why not Redux?

Overkill for single-user
Zustand is cleaner, faster
⏱️ Timer + Countdown
Native JS (setInterval)
Optional: date-fns
🌐 API Communication
Axios
3. 🧩 Backend Stack
🚀 Core
Node.js
Express.js
🗄️ Database
MongoDB
Mongoose
🔄 Dev Utilities
Nodemon
🧪 Validation (Optional but strong)
Joi

👉 Adds professionalism to backend

4. 📊 Analytics & Logic Layer
